# 臺北榮總特別休假日數試算系統 - 邏輯規則表 (若 P 則 Q)

本文件整理自試算系統原始碼之核心演算法，分為三個主要階段：(1) 週年制基礎日數判定、(2) 曆年制切段與進位邏輯、(3) 部分工時轉換邏輯。

---

## 階段一：週年制基礎給假判定 (對應勞基法第38條)
系統會先依據勞工的「累計年資」計算出該週年度應得的完整特休天數。

| 條件 (若 P ...) | 結果 (則 Q ...) | 備註 |
| :--- | :--- | :--- |
| **若** 累計年資為 `0.5` 年 (滿6個月未滿1年) | **則** 取得特休 `3` 日 | 請休期間為滿半年後至滿1年前 |
| **若** 累計年資大於等於 `1` 年，且未滿 `2` 年 | **則** 取得特休 `7` 日 | |
| **若** 累計年資大於等於 `2` 年，且未滿 `3` 年 | **則** 取得特休 `10` 日 | |
| **若** 累計年資大於等於 `3` 年，且未滿 `5` 年 | **則** 取得特休 `14` 日 | |
| **若** 累計年資大於等於 `5` 年，且未滿 `10` 年 | **則** 取得特休 `15` 日 | |
| **若** 累計年資大於等於 `10` 年 | **則** 取得特休 `15 + (年資 - 9)` 日 | 每年加給1日 |
| **若** 累計計算後之特休天數大於 `30` 日 | **則** 最高以 `30` 日計 | 依勞基法規定上限 |

---

## 階段二：曆年制比例切割與勞動部特殊進位規則
曆年制是將週年制的特休天數，依據該年度「重疊天數佔全年比例」進行切割。系統採用勞動部現行的「小數第二位特殊進位法」。

假設勞工到職日為 `M` 月 `D` 日，當年度週年制應得總天數為 `Days`：

| 條件 (若 P ...) | 結果 (則 Q ...) | 備註 / 公式對應 |
| :--- | :--- | :--- |
| **若** 需計算隔年1/1至到職週年前的「未休天數(遞延天數)」比例 | **則** 比例 = `((M - 1) + (D - 1) / 30) / 12` | 依據勞動部官方比例公式 |
| **若** 已算出上述比例 | **則** 原始未休數值 `rawValue` = `比例 × Days` | 算至小數點後多位 |
| **若** `rawValue` 的**小數點後第二位數字 > 1** | **則** 該未休天數進位至小數第一位 (即 `+0.1`) | *勞動部獨特規則*<br>例：4.7**2** -> 4.8 |
| **若** `rawValue` 的**小數點後第二位數字 <= 1** | **則** 該未休天數無條件捨去至小數第一位 | 例：4.7**1** -> 4.7 |
| **若** 已決定最終的隔年未休天數 (`unearned`) | **則** 當年度取得天數 (`earned`) = `Days - unearned` | 為避免浮點數誤差，用總數減去未休數 |
| **若** 需計算某特定「曆年度」的總特休天數 | **則** 總天數 = `(前一週年遞延的 unearned) + (當年度週年新產生的 earned)` | 即介面上的：<br>`calPrevEarned + calCurrEarned` |

---

## 階段三：部分工時 (時薪制) 轉換邏輯
若使用者選擇「部分時間工作勞工」，系統會以階段二算出的「曆年制天數」為基礎，轉換為「小時數」。
基準設定：全時勞工全年正常工作時間為 `2088` 小時 (52週 × 40小時 + 8小時)。

| 條件 (若 P ...) | 結果 (則 Q ...) | 備註 / 公式對應 |
| :--- | :--- | :--- |
| **若** 員工屬性為「部分工時」，且未輸入實際工時 | **則** 系統阻擋試算，提示「請輸入該年度結算之實際工作時數！」 | 必填欄位防呆 |
| **若** 實際工時 (`workHours`) < 0 或 > 4000 | **則** 系統阻擋試算，提示「請輸入合理的工作時數！」 | 數值防呆 |
| **若** 已得知實際工時 `workHours` 及全時特休天數 `Days` | **則** 時薪特休數 = `(workHours / 2088) × Days × 8 小時` | 依據《僱用部分時間工作勞工應行注意事項》 |
| **若** 上述時薪特休小時數計算完畢 | **則** 最終結果四捨五入至**小數點後第 1 位** | 顯示如：`10.7h` |
| **若** 員工屬性為「全時人員(月薪制)」 | **則** 略過階段三，直接輸出階段二的「天數」與圖表 | 不需轉換為小時 |
